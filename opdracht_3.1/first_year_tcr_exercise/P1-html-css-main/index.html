HTML/CSS

HTML wordt vaak gebruikt voor websites of browsers,
door zijn eenvoud is het gemakkelijk te
gebruik voor beginners tot de programmeerwereld.
Zoals het er nu uitziet, is CSS de beste manier om
te ontwerpen en kleur de website of browsers.
HTML staat voor: HyperText Markup Language.
CSS staat voor: Cascading Style Sheets.
HtML is als het wireframe van een website,
hiermee kunt u tekst, afbeeldingen,
audio en/of video's toevoegen,
wat betreft CSS, het geeft de wireframe vorm
en wordt gebruikt
om de stroom van hoe een website voelt
en beweegt te controleren.

Javascript

Javascript wordt vaak gebruikt in websites voor het onthouden

ook wordt het gebruikt voor video games zoals minecraft.
