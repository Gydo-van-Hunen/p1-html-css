About html/css

Waarom noem je de eerste HTML-pagina; index.html?

omdat het de inhouds pagina is (inhoudsopgave).
Daarom is de eerste pagina dus index.html.

Welke versies can html/css?

Op het momemt gebruiken we voor html, HTML5.
En voor css gebruiken we, CSS3.

Waarom is er verschil tussen elke website?

De reden tussen het verschil van elk website ligt in de css van een website.
Als de html het skelet van een website is, geeft de css het kleding om het er aantrekelijk te laten zien.
Daarom ziet elke website er anders uit. (een andere reden is dat het fysicaal past.)

Wat is de meest populaire browser?

De winnaar daarvan is google chrome, die wordt het meest gebruikt over de wereld.
Het staat bekend voor zijn snelheid en simple looks.
